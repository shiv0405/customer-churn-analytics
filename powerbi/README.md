# Power BI Assets

This starter includes actual working support assets for the report build:

- `../data/operations_kpis.csv` and `../data/incident_log.csv` sample the reporting model
- `measures.dax` contains starter DAX measures
- `semantic_model.json` and `dataset.metadata.json` describe the model
- `dashboard_preview.html` gives a browser-based preview of the KPI story

Latest preview week: 2026-03-25
The `.pbix` files remain transparent placeholders until the report is built in Power BI Desktop.
